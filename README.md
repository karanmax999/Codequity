# CodeQuity - Modern Tech Community Platform

A sleek, modern web platform for India's largest tech community, built with Next.js, TypeScript, and Tailwind CSS.

## ✨ Modern UI Features

### 🎨 Enhanced Visual Design
- **Glassmorphism Effects**: Modern backdrop-blur and transparency effects
- **Gradient Borders**: Beautiful gradient borders with hover animations
- **Enhanced Animations**: Smooth transitions, hover effects, and micro-interactions
- **Modern Color Scheme**: Sophisticated color palette with gradient accents

### 🚀 Interactive Components
- **Floating Action Button**: Expandable FAB with smooth animations
- **Feature Cards**: Interactive cards with gradient borders and hover effects
- **Enhanced Navigation**: Smooth hover animations and focus states
- **Progress Bar**: Animated scroll progress with shimmer effects

### 🎯 Animation System
- **Staggered Animations**: Sequential reveal animations for content
- **Hover Effects**: Lift, glow, and scale effects on interactive elements
- **Scroll Animations**: Intersection Observer-based reveal animations
- **Micro-interactions**: Subtle animations that enhance user experience

### 🎨 Design System
- **Custom CSS Classes**: Reusable utility classes for consistent styling
- **Glassmorphism**: `.glass` and `.glass-dark` classes for modern effects
- **Gradient Text**: `.gradient-text-*` classes for beautiful text effects
- **Hover Utilities**: `.hover-lift`, `.hover-glow` for interactive elements

## 🛠️ Tech Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 with custom animations
- **Components**: shadcn/ui with enhanced styling
- **Animations**: CSS animations with Intersection Observer

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🎨 Custom CSS Classes

### Glassmorphism
- `.glass` - Light glassmorphism effect
- `.glass-dark` - Dark glassmorphism effect

### Gradients
- `.gradient-text-blue` - Blue gradient text
- `.gradient-text-green` - Green gradient text
- `.gradient-text-purple` - Purple gradient text

### Animations
- `.animate-float` - Floating animation
- `.animate-pulse-glow` - Pulsing glow effect
- `.animate-slide-up` - Slide up reveal
- `.animate-fade-in` - Fade in effect

### Hover Effects
- `.hover-lift` - Lift effect on hover
- `.hover-glow` - Glow effect on hover
- `.focus-ring` - Enhanced focus ring
